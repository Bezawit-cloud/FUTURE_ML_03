Restaurant Reservation Chatbot

A restaurant reservation chatbot built using Dialogflow and Google Apps Script, connected to Google Sheets to record and manage bookings.

Features

Book a Table: Users can reserve a table by providing the date, time, and party size.

Cancel a Reservation: Users can cancel an existing booking.

View Menu: Users can ask about the restaurant menu.

Automatic Booking IDs: Each reservation is assigned a unique ID (e.g., BK-001).

Records Management: All reservations are stored in Google Sheets with status tracking (Active or Cancelled).

Technologies Used

Dialogflow: For NLP and intent recognition.

Google Apps Script: To handle webhook requests and update Google Sheets.

Google Sheets: To store booking data.

How It Works

User interacts with the chatbot.

Dialogflow recognizes the intent (Book_Table, Cancel_Booking, etc.).

Apps Script processes the request and updates Google Sheets.

Chatbot responds with confirmation, booking ID, or cancellation status."# FUTURE_ML_03"
